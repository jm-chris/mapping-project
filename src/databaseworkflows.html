The Database Workflows hub is a living collection of tutorials for China studies researchers who want to build, administer, and maintain datasets using modern computational tools. The modules are designed for researchers with high-level domain knowledge of East Asian history and languages who may be new to agentic workflows, Large Language Models, or programmatic data extraction.

These tutorials are semi-independent. You can enter the pipeline wherever it intersects with your current research needs — whether that means untangling the Unicode encoding of a Tibetan manuscript, extracting structured data from an archival finding aid, or publishing a searchable collection of historical documents. All modules use a shared sample dataset drawn from the Contemporary Chinese Village Gazetteer Data project at the University of Pittsburgh.

MODULE 01

Setup and Script Handling Crash Course

Introduces the shared CCVG dataset and covers the computational layer of working with East Asian materials: Unicode encoding pitfalls, the character simplification problem, and script-specific data challenges for Chinese, Tibetan, Mongolian, Manchu, and Uyghur.

Beginner 20 min No Coding
MODULE 02

OCR for Historical East Asian Materials

The image-to-text pipeline: preparing scans with ImageMagick, selecting the right OCR tool for your material (PaddleOCR, Transkribus, ABBYY, OCRmyPDF), and using LLM agents to post-correct output systematically.

Intermediate 45 min Command Line Python
MODULE 03

Extracting Structured Data: NER and Metadata

Pulling structured information from raw text: using LLM agents and specialized tools (WYD, MARKUS, spaCy) to extract document-level metadata and identify named entities in classical and modern Chinese.

Intermediate 45 min Python No-Code Options
MODULE 04

Calling Open Data APIs

Retrieving and transforming data from the APIs most relevant to China studies — CBDB, CHGIS, Wikidata/SPARQL — using an LLM coding agent to write and run the queries, without requiring prior programming experience.

Intermediate 40 min API Python
MODULE 05

Segmentation, Tokenization, and Normalization

Preparing text for analysis: modern vs. classical segmentation tools (Jieba, WYD), reconciling romanization variants across systems (Pinyin, Wade-Giles, Wylie), and standardizing messy historical datasets.

Intermediate 40 min Python No-Code Options
MODULE 06

Building and Publishing a Searchable Archive

The complete publication pipeline: designing a Dublin Core metadata schema, structuring a master CSV, and deploying a searchable collection using Omeka Classic and Apache Solr with the Smart Chinese Analyzer.

Advanced 60 min Omeka Command Line
